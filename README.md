# 📚 Smart Library System

A modern **Smart Library Management System** built using **React, Vite, and Tailwind CSS**, designed to streamline book management, user interaction, and library operations with a clean and responsive UI.

---

## Features

* Book browsing and management
* Dashboard with insights
* Search and filter functionality
* Fast and responsive UI
* Dark mode support (shadcn-style)
* Custom 404 Page handling

---

## 🛠️ Tech Stack

* **Frontend:** React + Vite
* **Styling:** Tailwind CSS (shadcn UI design system)
* **Routing:** React Router
* **State Management:** React Hooks
* **Build Tool:** Vite

---

## 📁 Project Structure

```
src/
│── components/        # Reusable UI components
│── pages/            # Application pages
│── lib/              # Utility and helper files
│── App.jsx           # Main app component
│── main.jsx          # Entry point
│── index.css         # Global styles
│
tailwind.config.js    # Tailwind configuration
vite.config.js        # Vite configuration
```

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/smart-library-system.git
cd smart-library-system
```

---

### 2️⃣ Install dependencies

```
npm install
```

---

### 3️⃣ Run the development server

```
npm run dev
```

---

### 4️⃣ Open in browser

```
http://localhost:5173
```

---

## 🧪 Available Scripts

```
npm run dev      # Start development server
npm run build    # Build for production
npm run preview  # Preview production build
```
